# JOSS Submission

This directory contains a paper draft and utilities for our submission
to the Journal of Open Source Software.

- `paper.md`: Paper draft
- `paper.bib`: References
- `performance_plot.py`: Script to generate performance comparison plot
- `performance_plot.pdf`: Performance comparison plot
- `timings.txt`: Saved output from performance tests
- `stylist.py`: Stylist input file for performance tests
- `fortran_rc_default.yaml`: Flint input file for performance tests
